MockMate-AI Mock Interview Assistant

The MockMate is an Agentic AI application that analyzes candidate resumes to generate personalized interview questions across technical, behavioral, and role-specific categories. It conducts interactive mock interviews using speech recognition and text-to-speech capabilities, recording and transcribing spoken responses. After evaluation, the system provides comprehensive feedback and generates a detailed PDF report with scores and improvement suggestions, all delivered through a Flask-based web interface.

Resume screening system

Developed an AI application to rank resumes based on job descriptions, utilizing NLP models like Sentence Transformers for embeddings and Langchain-Groq LLMs for summarization. Built a Streamlit interface for real-time resume comparisons. Enhanced matching accuracy using cosine similarity and TF-IDF-based keyword extraction. Improved processing speed by 30% with multiprocessing and designed robust pipelines for PDFs and DOCX formats using pdfplumber and python-docx. Integrated advanced preprocessing techniques, including lemmatization and stopword removal with SpaCy, to boost scalability and accuracy.

Flask/Streamlit NLP GROQ API
